About Qwen Town

What is Qwen Town?

Qwen Town is a 2D town simulation built with Python and FastAPI. It simulates a living, breathing community where NPCs (Non-Player Characters) work, trade, build, and interact with each other. The simulation runs on a tick-based system, updating the world state, economy, and NPC behaviors in real-time.

How It Works

The simulation is driven by a central tick loop that orchestrates various subsystems:

  • NPC Lifecycle: NPCs have needs (hunger, energy, happiness) and make decisions based on their state.
  • Economy: Resources are produced, traded, and priced dynamically based on supply and demand.
  • Buildings: Different building types provide services, production, or effects to the town.
  • Events: Random events like weather changes, disasters, or festivals impact the simulation.

Tech Stack

Backend

  • Python 3.11+
  • FastAPI (Web Framework)
  • SQLAlchemy 2.0 (ORM)
  • SQLite / PostgreSQL (Database)

Frontend

  • Jinja2 (HTML Templates)
  • HTMX (Dynamic Updates)
  • Tailwind CSS (Styling)
  • PixiJS (2D Rendering)
